
# Test typing speed game

## How to use
- to start just start typing and the timer will start to drop from 60 seconds to 0
- by pressing "ESC" you can change the displayed text or you can press the "reset" button
- by pressing "enter" you can restart the game and the text will remain the same
- you can see the statistics by pressing "View statsitics" button or clear them by pressing "Clear statistics"
